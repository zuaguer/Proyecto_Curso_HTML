Servicios principales de AWS

Servicios Disponibles Descipcion
Computo en la nube EC2

Funciones Lambda		 Es un servicio web que proporciona capacidad informática escalable bajo demanda en la nube, permitiendo a los usuarios crear y administrar instancias de servidores virtuales.

Lambda se define como un servicio de computación sin servidor que permite ejecutar código de forma automatizada en respuesta a eventos sin necesidad de aprovisionar o gestionar servidores ofreciendo una forma flexible y escalable de ejecutar código en la nube
Almacenamiento S3

Elastick Block Store		 Es un servicio de almacenamiento de objetos que ofrece escalabilidad, disponibilidad de datos, rendimiento y seguridad. Se utiliza para almacenar, proteger y recuperar cualquier cantidad de datos desde cualquier lugar en la web

Se utiliza para almacenar datos persistentes para instancias EC2, lo que indica que la información puede ser almacenada más allá del ciclo de vida de una instancia de EC2
Redes VPC

CloudFront		 Es una red virtual en la nube de manera aislada y personalizable, lo que significa que se puede definir las subredes que son el rango de direcciones IP dentro de la VPC

Se define como un servicio web que agiliza la distribución de contenido web estático y dinámico como archivos .html, .css, .js y archivos de imágenes a los usuarios.
